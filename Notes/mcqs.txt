Q3. What will be the output of the following code?
if 10 == 10:
    print("Welcome to CodingNinjas")
    
a)None x
b)Welcome to CodingNinjas 
Solution description
The code block in an if statement is executed if the boolean condition associated with the if statement evaluates to True. In this example, the boolean condition (10 == 10) evaluates to True and the code block is executed.
